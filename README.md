# Teoria dos Grafos: Tarefa 2

O projeto foi organizado em três diretórios:

- relatorio/: Guarda o relatório em formato pdf, bem como seu código-fonte
em LaTeX. Também abriga um diretório para as imagens que foram utilizadas.

- src/: Hospeda o código-fonte do programa, além do arquivo de palavras e
um simples script de compilação.

- scripts/: Aqui ficam os scripts utilizados para a geração das imagens, e
algumas figuras que não chegaram a ser usadas.
