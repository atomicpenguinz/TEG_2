O projeto está dividido em três diretórios.

O diretório "relatorio" possui o código-fonte LaTeX do relatório, o relatório em si
em formato pdf e uma pasta com as imagens presentes nele.

O diretório "src" contém o código fonte do programa, juntamente ao arquivo CSV
utilizado.

Já o diretório "scripts" contém os scripts e arquivos usados para a elaboração
das imagens.
